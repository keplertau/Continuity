# Archive

This folder is cold storage.

Use it for full session exports, old documents, retired drafts, bulky reference material, and historical context that may matter later but should not steer every future session by default.

Do not read this folder at `Start continuity` unless a current task needs exact history, evidence, or old source material.

## Suggested Folders

Create these only when useful:

```text
archive/
  sessions/
  documents/
  old-context/
  attachments/
```

## Naming

Use names that remain understandable later:

```text
archive/sessions/YYYY-MM-DD-short-title.md
archive/documents/YYYY-MM-DD-document-name.md
archive/old-context/YYYY-MM-DD-what-changed.md
```

## Rules

- Put full sessions here when they may be useful as evidence later.
- Put older documents here when they should be preserved but no longer steer current work.
- Add a pointer in `evidence.md` or `session-log.md` when archived material supports a current claim.
- Do not leave active decisions, current context, or live next steps only in the archive.
- If the archive contradicts active Continuity files, flag the conflict and check which record is newer or more authoritative.
