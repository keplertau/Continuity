# Decisions

This file holds decisions that should not be casually contradicted in future sessions.

Distinguish confirmed decisions from provisional leanings.

## Confirmed Decisions

Use this format:

- `YYYY-MM-DD` - Decision:
  Rationale:
  Constrains:
  Evidence:

## Provisional Leanings

Leanings are not decisions. Keep them separate so future AI sessions do not harden them accidentally.

Use this format:

- `YYYY-MM-DD` - Leaning:
  Why it seems plausible:
  What would change it:

## Superseded Decisions

Use this when a decision has been replaced.

- `YYYY-MM-DD` - Superseded:
  Replaced by:
  Why:
