## Visual Direction

Continuity's visual language should feel useful, warm, modern, and open. The project is not a database brand, a productivity gimmick, or a generic AI assistant wrapper. It is a continuity layer for human-AI work.

Core motifs:

- a continuous thread
- distinct context cards or registers
- human and AI workspaces connected without collapsing into one another
- visible order without sterile rigidity

Core palette:

- teal
- coral
- cream
- leaf green
- ink navy
- small yellow highlights

Avoid:

- generic infinity symbols
- robot mascots
- brain icons
- database cylinders
- purple-dominant AI gradients
- abstract memory clouds with no structure

The current raster assets were generated with Image 2.0 and then resized for GitHub.
